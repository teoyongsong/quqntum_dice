How this matches the circuit

The Python demo prepares |ψ⟩ ⊗ |ψ⟩ on six qubits: three per die, uniform over faces 001 … 110, then measures. In the browser we sample the same distribution: two independent uniform integers in 1…6, using crypto.getRandomValues when available.
